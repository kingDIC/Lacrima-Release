# Contributing

This repository is intentionally release-only.

It does not accept source-code contributions for the private Lacrima app, and it should not be used as a mirror of the private development repository.

## Appropriate Changes Here

Limited contributions may be appropriate for:

- public-facing release documentation
- repository hygiene
- distribution-facing metadata

## Not Accepted Here

- app source code
- gameplay features
- private implementation details
- source-level bug-fix pull requests

If you need to discuss development work, please use the private project channels maintained outside this repository.
