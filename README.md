# Lacrima Tears of Life Windows Releases

This public repository hosts Windows release artifacts for **Lacrima Tears of Life**.

It is a release-only operational counterpart to the private application repository. It does **not** contain the private source code, development history, or internal project materials for the app itself.

Platform scope for now: **Windows only**

Release channel: **stable only**

## What This Repository Is For

- Publishing public GitHub Releases for Windows builds
- Serving as the public download page for end users
- Hosting the release assets used by the app's auto-update flow

## What This Repository Is Not For

- Source code browsing
- Source-level bug triage
- Feature development
- Multiplayer or online-service coordination

Support, security handling, and source development are managed separately from this repository.

## Where To Download

Download the current Windows build from the repository's **Releases** page.

Each shipped version is published as a GitHub Release tagged in the form:

`vX.Y.Z`

## Release Asset Contract

Every shipped Windows release is expected to include these assets:

- Installer `.exe`
- `RELEASES`
- `*-full.nupkg`

An additional delta package may also be present if generated by the release pipeline, but it is optional.

These files exist to support both direct downloads and in-app updates for the stable Windows channel.

## Auto-Updates

The Electron app uses this repository's GitHub Release assets as its Windows update feed target.

For stable updates to work reliably:

- release tags should remain in `vX.Y.Z` format
- required assets must be attached to each shipped version
- previously published update assets should not be deleted or replaced
- the repository owner/name should remain stable after users have installed the app

Further operational guidance is documented in [docs/release-policy.md](/d:/Programmieren/Lacrima-Release/docs/release-policy.md).

## Unsigned Windows Warning

Windows builds are currently **unsigned**.

That means some users may see warnings from:

- Microsoft Defender SmartScreen
- web browsers
- Windows download protection prompts

This is expected for now. Users should download only from this repository's official Releases page and verify they are using the intended release.

## Scope

This repository is intentionally minimal and public-facing. It exists to distribute official Windows release artifacts for the stable channel only.
