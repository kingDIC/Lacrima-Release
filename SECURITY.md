# Security Policy

## Scope

This repository is a **distribution-only** public repository for Windows release artifacts of **Lacrima Tears of Life**.

It is not the private source repository and should not be used to disclose source-sensitive details, internal architecture, or private code excerpts.

## Do Not Report Security Issues Publicly Here

Please do **not** open public GitHub issues in this repository for suspected vulnerabilities, exploit details, private implementation notes, or sensitive reports.

Public posting in this repository may create unnecessary exposure for a private codebase that is intentionally not mirrored here.

## Reporting Contact

Please report security concerns through the private contact path below:

- Security contact: `[replace-with-security-contact]`
- Secondary contact: `[replace-with-secondary-security-contact]`
- PGP or encrypted reporting details: `[replace-with-encryption-details-if-used]`

## What To Include

When reporting a security issue, include:

- affected version
- reproduction steps if known
- impact summary
- any relevant screenshots or logs that do not expose unrelated private data

## Repository Boundary

This repository only hosts release artifacts and release-facing documentation.

Security fixes, source investigation, and internal remediation are handled outside this public repository.
